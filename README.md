## Assignment 1: Java Login System

## Description
This project is a simple Java-based login system developed as part of the BSD 214 (Object Oriented Programming 2) course.
The system prompts a user to enter a username and password, masks the password input using asterisks (*), and allows a
maximum of three login attempts.

## Features
- Username and password authentication
- Password masking during input
- Maximum of three login attempts
- Clear success and error messages
- Implemented using Java and OOP principles

## Technologies Used
- Java
- Object-Oriented Programming concepts
